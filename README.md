코드 블럭 3개는 백틱(물결 따옴표)3개로 감싸주자
```python
print("Hello")
if x < 10:
    print('x는 10보다 작습니다.')
else:
    print('x는 10보다 같거나 큽니다.')
```

한 줄만 코드로 표시하고 싶을 때
=> 백틱 하나로 감싸 주면 됨
